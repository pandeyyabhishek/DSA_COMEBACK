# 🔥 What is Two Pointer Approach?

👉 Instead of using nested loops (**O(n²)**),
we use **two indices** moving smartly to reduce complexity to **O(n)**.

```
i → starting pointer
j → ending pointer
```

Pointers move based on condition.

---

## 🧠 Core Idea

👉 Use **sorted nature** OR **continuous window**.

You always ask:

> Can I adjust answer by moving only one pointer?

If YES → Two Pointer.

---

# ✅ Types of Two Pointer Patterns (VERY IMPORTANT FOR PBC)

---

# 1️⃣ Opposite Direction Pointers

Most classic pattern.

```
i = 0
j = n-1
```

Move inward.

---

## Pattern Logic

```
while(i < j){
   if(condition satisfied) → answer
   else if(sum too small) i++
   else j--
}
```

# 2️⃣ Same Direction (Fast & Slow Pointer)

Also called:

✅ Floyd's algorithm
✅ Runner technique

```
slow → moves 1 step
fast → moves 2 steps
```

---

## Used When?

* Detect cycle
* Remove duplicates
* Partition array
* Rearrange data

---

## 🔥 Questions

* Linked List Cycle
* Middle of Linked List
* Remove Duplicates
* Move Zeroes
* Sort Colors (Dutch National Flag)

⚡ VERY HIGH INTERVIEW + PBC frequency.

---

# 3️⃣ Sliding Window (Dynamic Two Pointer)

⭐ MOST IMPORTANT PATTERN FOR PBC

Window expands & shrinks.

```
left ----- right
```

---

## When to use?

👉 Subarray
👉 Continuous segment
👉 Maximum / Minimum length

---

## Template

```
while(right < n){
    include right element

    while(window invalid){
        remove left element
        left++
    }

    update answer
    right++
}
```
