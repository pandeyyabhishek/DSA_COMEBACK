# 🔥 Step 2: Learn Core Pointer Patterns

These are the REAL syllabus.

---

## 1️⃣ Traversal Pattern

Used in:

* search element
* length
* printing
* sum

Logic:

```
Move curr until NULL
```

---

## 2️⃣ Two Pointer Pattern (VERY IMPORTANT)

### Slow + Fast Pointer

```
slow = head
fast = head
```

Move:

```
slow -> 1 step
fast -> 2 step
```

Used in:

* Middle of LL
* Cycle detection
* Palindrome
* Remove nth node
* Split list

👉 Most asked pattern.

---

## 3️⃣ Reverse Pattern ⭐⭐⭐⭐⭐

Interview KING.

### Core Logic:

```
prev = NULL
curr = head

while(curr){
   next = curr->next
   curr->next = prev
   prev = curr
   curr = next
}
return prev
```

Used everywhere:

* reverse list
* palindrome
* reverse k group
* reorder list
* rotate list

👉 Master once → many questions solved.

---

## 4️⃣ Dummy Node Pattern ⭐⭐⭐⭐⭐

Top interviewer trick.

Create fake node:

```
dummy -> head
```

Used when:

* deleting nodes
* merging lists
* head may change

Example:

* Remove Nth node
* Merge two lists
* Partition list

👉 Removes edge cases.

---

## 5️⃣ Fast Pointer Gap Pattern

Keep distance between pointers.

```
fast moves k steps first
then move slow + fast together
```

Used in:

* Remove Nth from end
* Find kth from end

---

## 6️⃣ Cycle Pattern

Use Floyd’s Algorithm.

```
slow = 1 step
fast = 2 step
if meet → cycle exists
```

Questions:

* Detect cycle
* Find cycle start

---

## 7️⃣ Merge Pattern

Used in:

* Merge 2 sorted lists
* Merge K lists

Logic:

```
compare nodes
attach smaller
move pointer
```

---

## 8️⃣ Reversal Variations Pattern

Hard questions are only modified reverse.

Examples:

* Reverse between L and R
* Reverse K group
* Reorder list

👉 Understand boundary handling.
