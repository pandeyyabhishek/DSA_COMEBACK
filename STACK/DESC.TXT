# 1️⃣ Stack Concept (Core Idea)

👉 **LIFO (Last In First Out)**

Like plates:

```
Push → add element
Pop  → remove top element
Top  → see top element
```

### Operations

| Operation  | Meaning     |
| ---------- | ----------- |
| push(x)    | insert      |
| pop()      | remove      |
| peek/top() | get top     |
| empty()    | check empty |

---

## Implementation Types

1. Array
2. Linked List
3. STL Stack (C++ / Java Stack / Deque)

👉 Interviews rarely ask implementation now.
They test **STACK THINKING PATTERNS**.

# ✅ Pattern 2 — Monotonic Stack (🔥 MOST IMPORTANT)

This is **THE REAL STACK TOPIC**.

---

## What is Monotonic Stack?

Stack maintains elements in:

* Increasing order
  OR
* Decreasing order

---

### WHY?

To find quickly:

✅ Next Greater
✅ Next Smaller
✅ Previous Greater
✅ Previous Smaller

---

## Template (VERY IMPORTANT)

```java
Stack<Integer> st = new Stack<>();

for(int i=0;i<n;i++){
    while(!st.isEmpty() && arr[st.peek()] < arr[i]){
        st.pop();
    }
    st.push(i);
}
```

👉 **While loop = key**
